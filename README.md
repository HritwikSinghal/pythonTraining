# file based key value store

## Installation.

1. #### Clone this repo
    ```sh
    git clone --depth 1 -b master </URL>
    ```

2. #### Install requirements.

    ```sh
    pip3 install -r requirements.txt
    ``` 

#### Running.

- cd into the directory where the program is cloned
- ```sh
  python3 start.py --help   
  ```
